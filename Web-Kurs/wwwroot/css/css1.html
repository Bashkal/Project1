CSS Seçicileri

CSS seçiciler, stilize etmek istenilen elementi bulmak veya seçmek için kullanılır

Css seçileri beş kategoride inceleyebiliriz

  • basit seçiciler(ad kimlik sınıfa göre öge seçimi)
  • sözde sınıf seçiciler(belirli bir duruma göre seçim için)
  • sözde eleman seçiciler(bir ögenin bir parçasını seçer ve stilize eder)
  • öznitelik seçiciler(bir öznitelik veya öznitelik değerine göre seçim yapar)
  • kombinatör seçiciler(aralarındaki belli bir ilişkiye yönelik seçim)
